# Cookiecutter Python Package Template

This is a basic template with minimal features.

## Features

* Pipenv-oriented
* pre-commit hooks
* black
* isort
* flake8

## Setup

Install [cruft](https://github.com/timothycrosley/cruft/) or [cookiecutter](https://github.com/cookiecutter/cookiecutter)

```
cruft create <url>
```

or

```
cookiecutter <url>
```
